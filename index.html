KNOWLEDGE

HTML

HTML language (hypertext markup language) is used for the development and creation of web pages. It is made up of a series of tags that the browser interprets. Among the tags that are included within the HTML language are: hyperlinks, image tags, page breaks, among others.

CSS

CSS stands for "cascading style sheets". Basically, it is a language that handles the design and presentation of web pages, that is, how they look when a visitor visits them. It works together with the HTML language, which takes care of the basic content of the pages

Javascript

JavaScript is the only programming language that works natively in browsers (interpreted language without the need for compilation). Therefore it is used as a complement to HTML and CSS to create web pages

React

Python

Python is a fast, scalable, robust and open source writing language, advantages that make Python a perfect ally for Artificial Intelligence. It allows you to capture complex ideas with a few lines of code, which is not possible with other languages.

Node.js

Node. js es un entorno JavaScript que nos permite ejecutar en el servidor, de manera asíncrona, con una arquitectura orientada a eventos y basado en el motor V8 de Google. Es una plataforma que avanza muy rápidamente y cada vez está más presente en el mercado.
